# Пульт охраны банка

## Как запустить сайт

Сайт подключается к базе данных и позволяет:
- посмотреть активные пропуска
- узнать, кто сейчас в хранилище
- увидеть длительность визитов
- посмотреть историю посещений сотрудника

### 1. Клонируйте репозиторий

```bash
git clone <repo_url>
cd <repo_folder>
```

### 2. Создайте виртуальное окружение

```bash
python -m venv venv
```

Активируйте окружение.

```bash
source venv/bin/activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Создайте файл `.env`

В корне проекта создайте файл `.env` и добавьте в него параметры:

```env
DEBUG=True
DB_HOST=хост
DB_PORT=порт
DB_NAME=название базы
DB_USER=пользователь
DB_PASSWORD=пароль
SECRET_KEY=секретный ключ
```

### 5. Запустите сервер

```bash
python manage.py runserver 0.0.0.0:8000
```

### Откройте в браузере

```text
http://0.0.0.0:8000/
```

## Ожидаемый результат

- отображается список активных пропусков
- доступна страница со списком людей в хранилище
- при клике на пропуск открывается история визитов сотрудника
- для неверного пропуска возвращается страница 404
